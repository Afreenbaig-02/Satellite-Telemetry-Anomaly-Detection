# Satellite Telemetry Anomaly Detection

## Overview
This project detects anomalies in satellite telemetry data using Machine Learning. 
The system analyzes satellite sensor values like temperature, voltage, and signal strength 
to identify abnormal behavior that may indicate system issues.

## Technologies Used
Python  
Pandas  
NumPy  
Scikit-learn  
Matplotlib  

## Dataset
The dataset contains simulated satellite telemetry values:
- Temperature
- Voltage
- Signal Strength

File used: satellite_telemetry.csv

## Machine Learning Model
Isolation Forest algorithm is used to detect anomalies in telemetry data.

## Output
The model identifies abnormal readings and visualizes them using a scatter plot.
